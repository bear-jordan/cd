# Smart CD
Simple cd utility to move up a specified number of directories.


## Usage

1. Move script to the user's local bin (e.g. `~/.local/bin/smart-cd.sh`)
```sh
mv ./smart-cd.sh ~/.local/bin/smart-cd.sh
```

2. Update .zshrc

```sh
source ~/.local/bin/smart-cd.sh
```

3. Reload .zshrc

```sh
source ~/.zshrc
```

4. Use the script

```bash
cd.. 2
```
